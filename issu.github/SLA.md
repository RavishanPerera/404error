# Service Level Agreement (SLA)

We aim to respond to issues in a timely manner:

- Bug Reports:
  - Response: within 24 hours
  - Fix: within 72 hours

-  Feature Requests:
  - Response: within 48 hours
  - Decision/Planning: within 7 days

- Technical Debt:
  - Response: within 48 hours
  - Resolution: based on sprint planning
